# 골든셋 (09문서 1-3)

이 디렉터리의 `*.json`은 실제 whisper 검증 경로 실행 결과에서 자동 추출한
**후보**다. 사람이 오디오를 직접 들으며 검토하기 전까지는 회귀 채점의
정답지로 쓰지 않는다 — 각 항목의 `status` 필드가 `unreviewed`인 동안은
`eval_golden.py`가 경고를 출력하고 채점에서 제외한다.

## 검토 방법

1. `video_id`로 유튜브 영상을 열고 `sec`로 이동.
2. `quote`가 실제 발언과 일치하는지 확인.
   - 일치 → `status: "confirmed"`로 변경.
   - 실제로는 다른 말(할루시네이션) → `status: "hallucination"`으로 변경하고
     `correct_quote`에 실제 발언을 채운다(선택).
   - 애매함(잔향/겹침으로 판단 불가) → `status: "ambiguous"`.
3. 이 영상에서 **검증 게이트가 놓친** 실제 코치 발언이 있다면
   `missed_feedbacks[]`에 손으로 추가한다(재현율 측정에 필요).
