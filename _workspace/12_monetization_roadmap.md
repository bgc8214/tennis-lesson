# 12. 수익화 로드맵 — 제품 준비부터 B2B 확장까지

작성: 오케스트레이터 세션 / 2026-07-17
성격: **사업/GTM(Go-To-Market) 전략 문서** — 기술 로드맵은 11문서, 기능 기획은 09문서 참조
전제: 1인 개발, 현재 도그푸딩 단계, PMD 수익모델(Free/베이직 4,900원/프로 9,900원/B2B) 유효

## 0. 전체 그림 (5단계)

```
[Stage 0] 제품·사업 준비 ──> [Stage 1] 클로즈드 베타 ──> [Stage 2] 오픈 베타·커뮤니티 런칭
                                                              │
[Stage 4] B2B 코치 확장 <── [Stage 3] B2C 유료화 <────────────┘
```

각 단계는 **통과 조건(게이트)**을 만족해야 다음으로 넘어간다.
실패 신호가 나오면 다음 단계로 돈/시간을 태우지 않는 것이 1인 개발의 생존 원칙.

## Stage 0. 제품·사업 준비 (지금 ~ 4주)

### 0-1. 품질 신뢰선 (기술)

- 골든셋 재현율 목표치 설정 후 통과할 때까지 출시 금지. 제안:
  - 코치 핵심 발언 재현율 ≥ 70% (골든셋 4개 영상 기준)
  - 잘못된 카드(검증 통과했지만 의미 왜곡) = 사람 검수 기준 0건
  - 처리 시간: 60분 영상 기준 목표치 실측 후 확정
- 근거: 이 서비스의 첫인상은 "AI가 내 레슨을 정확히 알아들었는가" 단 하나.
  첫 리포트가 틀리면 재방문 없음 (KPI Clarity Score의 존재 이유).

### 0-2. 법적 리스크 정리 — 서비스화 전 필수 확인

- **YouTube TOS**: 서버에서 yt-dlp로 다운로드하는 행위는 유저 본인 영상이라도
  YouTube 약관 위반 소지가 있음. 개인 도그푸딩과 상용 서비스는 법적 노출이 다름.
  → **상용화 시점에는 Phase 3(직접 업로드)를 기본 동선으로** 하는 것이 품질/비용
  문제이자 동시에 법적 리스크 해소책 (11문서 인사이트 ③과 정합).
  유튜브 링크 경로는 베타 기간 한정 or 자막 기반으로 축소 검토.
- **개인정보/초상권**: 레슨 영상에는 코치·타인 얼굴/음성 포함 → 개인정보처리방침,
  "본인이 권리를 가진 영상만 업로드" 약관 명시, 음성 데이터 즉시 파기(무저장) 명문화
  — 무저장 파이프라인이 마케팅 포인트이자 컴플라이언스 포인트.
- **사업 절차**: 사업자등록(간이/일반) → 유료화 전 통신판매업 신고 → 구독 결제 시
  토스페이먼츠/포트원 심사에 둘 다 필요. 지금 미리 해두면 Stage 3에서 병목 없음.

### 0-3. 인프라 확정 (어디에 올릴 것인가)

| 구성 | 선택 | 이유/비용 |
|---|---|---|
| 프론트 | **Vercel** (Next.js 표준) | 무료 티어로 시작, 커스텀 도메인 무료. 트래픽 증가 시 Pro $20/월 |
| 백엔드 API | **Cloud Run 유지** | 이미 구축됨. `--no-cpu-throttling` + min-instances=0 |
| DB/인증 | **Supabase 유지** | 무료 티어 → 유저 늘면 Pro $25/월 |
| 분석 파이프라인 | 베타: server-paid(프록시+Groq, 회당 ~130원) 또는 로컬 worker(0원) → 상용: **Phase 3 직접 업로드(0원)** | 10/11문서 |
| 도메인 | `.kr` 또는 `.app` 구매 (연 1~3만원) | 카톡 공유 링크 신뢰도에 도메인이 중요 |
| 고객 응대 | **카카오톡 채널** (무료) + 채널톡(추후) | 타겟 유저가 카톡 생활권 |
| 모바일 | **PWA 우선** (홈 화면 추가 유도) → Stage 3 이후 네이티브 앱 검토 | 앱스토어 수수료 30% 회피 + 1인 개발 리소스. 단, Phase 3 촬영-업로드 UX가 커지면 앱 필요성 재평가 |

### 0-4. 최소 제품 완성 항목 (09문서에서 선별)

- 연습 체크리스트(#2), 하이라이트 클립 카톡 공유(#4) — 리텐션/바이럴의 최소 세트
- 카드별 👍/👎 (Clarity Score 측정 인프라 — Stage 1 게이트 판정에 필요)
- 온보딩: 첫 방문자가 30초 안에 가치를 보도록 **샘플 리포트 1개 공개** (로그인 전 열람)

**Stage 0 통과 조건**: 골든셋 목표 달성 + 약관/개인정보처리방침 게시 + 결제 전 단계 사업 절차 완료

## Stage 1. 클로즈드 베타 (4~8주, 10~30명)

### 대상 모집 (돈 안 쓰는 채널만)

1. **본인 테니스 동호회/레슨 동기** — 최우선. 직접 시연하고 카톡으로 리포트 전달
2. 본인 코치 1~2명에게 리포트 보여주기 — B2B 씨앗 + 정확도 검증자
3. 지인의 지인 (동호회 오픈카톡방 1~2곳, 홍보 아닌 "만들었는데 써볼 사람" 톤)

### 측정할 것 (이 단계는 매출이 아니라 데이터가 목적)

| 지표 | 목표 | 의미 |
|---|---|---|
| 주간 리텐션 (다음 레슨 후 재사용) | ≥ 40% (PMD 목표 60%의 베타 버전) | 제품이 습관이 되는가 |
| Clarity Score (👍 비율) | ≥ 70% | 리포트가 실제로 맞는가 |
| 카톡 공유율 (리포트당) | ≥ 20% | 바이럴 루프 가능성 |
| 코치 반응 | 정성 인터뷰 3건+ | B2B 가설 검증 |

**Stage 1 통과 조건**: 리텐션·Clarity 목표 달성. 미달 시 → 09문서 품질 항목으로
회귀, 오픈 베타 연기. (여기서 밀어붙이면 마케팅 비용이 밑 빠진 독)

## Stage 2. 오픈 베타 · 커뮤니티 런칭 (8~12주)

### 2-1. 채널별 마케팅 세분화 (한국 테니스 시장 특화)

| 채널 | 방법 | 비용 | 기대 효과 |
|---|---|---|---|
| **네이버 대형 테니스 카페** | 광고 아닌 **후기/사용기 형식** 글 ("레슨 복기 이렇게 하니 달라졌다" + 실제 오답노트 캡처). 카페 규정 확인 후 운영진 제휴 문의도 병행 | 0원 | 국내 테니스 담론의 중심. 최우선 채널 |
| **인스타그램** | 릴스: "레슨 1시간 → 오답노트 30초" Before/After 포맷. #테니스레슨 #테니스스타그램 #테린이 태그권. 계정은 브랜드 계정보다 **개발자 본인의 빌딩 인 퍼블릭** 톤이 초기엔 효과적 | 0원 (광고는 Stage 3부터) | 2030 테니스 붐 코어층 |

### 2-2. 콘텐츠 전략의 핵심

**리포트 자체가 광고다.** 카톡으로 공유된 오답노트가 받는 사람에게 "나도 하고
싶다"를 유발하는 구조 — 공유 리포트의 디자인/CTA 품질에 마케팅 예산보다 먼저 투자.

### 2-3. 측정

- 주간 신규 가입, 채널별 유입(UTM), 가입→첫 리포트 완료율(활성화), 주간 리텐션
- **Stage 2 통과 조건**: 오가닉 신규가 주 20명+ 안정 유입, 주간 리텐션 50%+,
  월 서버비가 감당 범위(무료 티어 + 프록시 소액) 이내

## Stage 3. B2C 유료화 (오픈 베타 안정 후)

### 3-1. 가격 (PMD 모델 검증 후 미세조정)

| 등급 | 가격 | 제공 | 심리적 프레임 |
|---|---|---|---|
| Free | 0원 | 월 3회 복기 | 습관 형성 최소량 (주 1회 레슨 유저가 4주 중 3주 커버 → 4주째에 결제 동기) |
| 베이직 | 월 4,900원 | 무제한 복기 + 고질병 트래커(09문서 #1) | **"레슨 1회 10만원의 5%로 레슨 전부를 자산화"** — 이 한 줄이 핵심 카피 |
| 프로 | 월 9,900원 | + Vision 관절 분석 (Phase 2) | Phase 2 완성 전까지 판매 보류 — 빈 약속 금지 |

- 유료화 트리거 기능은 **고질병 트래커**(누적 데이터 기반이라 이탈 비용 발생 =
  구독 유지 장치). 무제한 횟수만으로는 약함.
- 연간 플랜(2개월 할인)은 리텐션 데이터 확인 후 도입.

### 3-2. 결제 인프라

- **토스페이먼츠 또는 포트원(아임포트)** 정기결제(빌링키) — 국내 카드 구독 표준.
  사전 요건: 사업자등록 + 통신판매업 신고 (Stage 0에서 완료했어야 함)
- 첫 구현은 단순하게: 인앱 크레딧 충전(단건 결제)로 시작 → 구독 전환 검증 후
  빌링 도입도 가능 (기존 크레딧 시스템 재활용)

### 3-3. 단위 경제 (반드시 흑자 구조 확인 후 광고비 집행)

```
베이직 유저 1명/월: 매출 4,900원
원가(월 8회 복기 가정):
  - server-paid 경로: 8 × 130원 ≈ 1,040원 → 마진 ~79%
  - Phase 3 직접 업로드: ≈ 0원 → 마진 ~97% (결제 수수료 제외)
```
→ **Phase 3 완성이 곧 마진 개선 프로젝트.** 유료 유저 100명 시점의 월 매출
49만원 기준, 원가 차이는 월 10만원+ = Phase 3 개발의 직접적 ROI.

### 3-4. 이 단계에서 시작하는 유료 마케팅

- 인스타/네이버 검색 광고 소액 테스트 (월 10~30만원 상한), CAC < 3개월 LTV 확인
  전까지 증액 금지
- 추천인 제도: 친구 초대 시 양쪽 무료 크레딧 (카톡 공유 루프 강화)

**Stage 3 통과 조건**: 무료→유료 전환율 3%+ / 월 구독 이탈률 10% 미만 / 단위 경제 흑자

## Stage 4. B2B 코치·아카데미 (B2C 안정 후)

PMD 8.2의 구현. **코치는 유저이자 유통 채널** — 코치 1명 = 수강생 20~50명.

1. **입구**: 코치 확인 링크(09문서 #5) — B2C 유저가 리포트를 코치에게 보내는
   순간이 영업 접점. 코치용 화면에 "내 수강생 전체 관리하기" CTA
2. **코치 대시보드** (신규 개발): 수강생별 리포트 히스토리, 고질병 추이,
   레슨 전 브리핑 자동 발송
3. **가격 가설**: 코치당 월 19,000~29,000원 (수강생 무제한) 또는
   아카데미 단위 계약. 코치가 수강생에게 "복기는 이 앱으로 하세요"라고 말하는
   순간 CAC가 0원이 되는 구조
4. **파일럿**: Stage 1에서 관계 맺은 코치 2~3명과 무료 파일럿 → 사례 확보 →
   테니스 코치 커뮤니티/연수 채널로 확장

**게이트**: 파일럿 코치의 수강생 활성률이 B2C 오가닉보다 높은지 검증.

## Stage 5. 확장 (선택지 메모만)

- **종목 확장**: 파이프라인은 종목 불문 (용어 사전 + 프롬프트 교체 수준).
  골프는 레드오션, **배드민턴/탁구/필라테스·PT**가 동일 페인포인트(고가 레슨
  + 휘발성 피드백) 보유. B2C보다 B2B(강사) 진입이 유리할 수 있음
- **글로벌**: Whisper/Gemini 다국어 지원으로 기술 장벽 낮음. 일본(테니스 스쿨
  문화 강함) 우선 검토
- 이 단계 의사결정은 Stage 3~4 데이터 이후로 미룸 — 지금 고민 금지
